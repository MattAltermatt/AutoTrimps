# 🎨 Design mockups

Persisted design mockups — committed so they survive dev-server restarts and
`dist/` regeneration. Open the `.html` files directly in a browser (they're
self-contained), or view the `.png` renders inline on GitHub.

**Convention:** name `YYYY-MM-DD-<topic>.html` (+ optional `.png` render). Save
the mockup here and commit it in the same change as the spec/work it illustrates.
Keep the source HTML, not just a PNG — a PNG can't be re-rendered with tweaks.
